# 🪸 Reef Siege — Deep-Sea Defense

An endless undersea tower-defense game built for **playing while you multitask** — on desktop
or phone. Towers auto-fight, your shoal auto-resummons, heroes auto-cast. A re-themed sister
game of [Castle Siege](https://github.com/dangsteve/STAT184).

**Play:** open `index.html` locally, or enable GitHub Pages (Settings → Pages → Deploy from a
branch → `main` → `/ root`) and play at `https://dangsteve.github.io/reef-siege/`.

## The deep

- **3 reefs**: Lagoon Shallows (Easy), Kelp Crossing (Medium, 2 currents), The Abyssal Trench
  (Hard, 3 currents, bioluminescent dark).
- **9 towers** — Harpoon Post, Anchor Mortar, Ice Coral, Thermal Vent, Great Harpoon, Urchin
  Lab, Eel Spire, Treasure Chest, Pearl Beacon. Badged ⛰＋✈ (hits free-swimmers) or ⛰.
- **12 shoal troops** — Jellyfish, Pistol Shrimp, Crab Guards, Swordfish, Pufferfish Darters,
  Mantis Shrimp, Snail Shellguards, Cuttle Mages, Angelfish Healers, Seahorse Riders, Turtle
  Paladins, Colossal Crabs.
- **6 heroes** — Porous Pete the Fry-Cook (twin spatulas, from wave 1), Coralie Swiftfin,
  Inkwell the Deep Sage, Sister Marina, Crushclaw the Crab Jarl, Moray the Shadow Eel.
- **3 Sea Legends** rescued from Locker Wardens, kept forever in your Vault:
  **King Neptune**, **The Flying Dutchman**, and **Davy Jones**.
- **Spells**: Boiling Geyser, Healing Current, and **MAELSTROM** — the last-minute clutch.
- Wheel of Fortune every new run, Golden Sea Turtle treasure events, peak-build restarts,
  full touch/phone support.

All art procedural canvas, music generated live with WebAudio — an original fan-made undersea
world drawing on public-domain sea folklore. No external assets.
